Passo Passo Desafio TOTVS
- Instalar o Angular Globalmente
  npm i g @angular/cli

- Criar projeto Angular
  ng new spa-desafio-totus

- Instalar Design System (Angular Material) 
  ng add @angular/material

- Utilizando template do Angular Material
  (Criar componente Dashboard)
  ng generate @angular/material:dashboard dashboard

  (Criar componente Lista Entregas)
  ng generate @angular/material:table lista-entregas
